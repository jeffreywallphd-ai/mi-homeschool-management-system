## Terms

| Term | Meaning |
| --- | --- |
| Activity log | A parent-owned record of educational activity, project work, reading, fieldwork, practice, or instruction. |
| Course | A planned high-school learning unit that may earn credit and map to one or more requirement areas. |
| Course completion | A domain decision that a student has completed a course according to parent-defined standards. |
| Course description | A structured explanation of course title, content, resources, methods, assessment, credit value, and requirement mappings. |
| Credit award | Parent decision granting academic credit for completed coursework. |
| Curriculum plan | The parent's intended educational plan for a course, subject, term, or school year. |
| Diploma | A family-issued credential based on parent-defined graduation standards. |
| Evidence tag | A label connecting an artifact to a course, requirement area, skill, or portfolio category. |
| Generated document | A rendered report, transcript, diploma, packet, or export produced from source records. |
| Graduation packet | A family archive/export containing transcript, diploma copy, course descriptions, graduation plan, portfolio index, and selected evidence. |
| Graduation plan | The parent-defined standard for graduation, including required credits and categories. |
| Official family-issued record | A report card, transcript, diploma, or packet issued by the homeschool family. |
| Portfolio artifact | A file, note, photo, project, test, writing sample, or external record kept as educational evidence. |
| Progress evaluation | Parent-authored or system-assisted summary of progress over a period. |
| Requirement area | A legal, guidance, or parent-defined subject area used for coverage tracking. |
| Requirement mapping | A link between a course or evidence item and a requirement area. |
| School profile | Parent-defined homeschool administrative identity used on official family-issued records. |
| Stored file | A locally stored file with metadata, category, ownership, and checksum. |
| Transcript | A high-school course history with grades, credits, GPA, graduation information, and parent/school signature block. |

## Language Rule

Use "parent-issued," "family-issued," "records show coverage," and "parent-defined graduation standards" when describing official records. Avoid language that implies state approval, accreditation, or legal certification.
