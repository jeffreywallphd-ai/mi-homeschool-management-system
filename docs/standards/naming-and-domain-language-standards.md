## Naming Rules

- Use domain terms from the glossary.
- Prefer "parent," "student," "course," "credit award," "graduation plan," "portfolio artifact," and "generated document."
- Use "family-issued" or "parent-issued" for official homeschool records.
- Use "requirement area" rather than "compliance rule."

## Avoid

- Compliance certification language.
- State approval language.
- Generic school-SIS terms when homeschool-specific terms are clearer.
- Abbreviations that obscure meaning for a nontechnical parent.

## UI Language

UI text should be plain and specific. It should show actions and record status without implying legal conclusions.
