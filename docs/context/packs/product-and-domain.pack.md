## Must Preserve

- The app is records-and-credentials centered.
- Parent is the homeschool administrator.
- Student workflows are allowed but high-stakes finalization remains parent-owned.

## Common Failure Modes

- Reducing the system to courses, assignments, and grades only.
- Using public-school SIS language where homeschool language is clearer.
