## Purpose

This directory documents legal-reference material that informs the system's requirement model and wording. It is not legal advice.

## Rules

- Michigan is the first seeded jurisdiction profile.
- Legal text and MDE guidance must be represented as source-backed reference data, not as hard-coded application assumptions.
- The app must help parents maintain records and show selected subject-area coverage.
- The app must not certify legal compliance, accreditation, state approval, or MDE acceptance.
